About the Examples

The examples are based on the RTT examples available at the Orocos Project website. We have refactored the examples to not only illustrate the concepts of RTT but also of BRIDE.

Downloading the Examples

The latest examples can be downloaded from:

RTT Examples for BRIDE

The examples are in the form of Eclipse projects that can be imported directly to your workspace from the compressed archive file.

Running the examples

We assume a standard Orocos installation without ROS.

After building the project(s) open a terminal and do:

  1. $ cd [example projext directory] (e.g. rtt-execution/ ).
  2. $ sudo make install
  3. $ deployer-gnulinux deployment/[example project name].cpf (e.g. rtt-execution.cpf).

Note: Remember to make unintall and then make install after edting the examples.
