Adding an OutputPort

This is how a TaskContext with an OutputPort looks like when it is first created in the model diagram:

The easiest way to add a OutputPort or any other model element is to select, drag and drop the element from the Palette to the diagram.

  1. Select a OutputPort.
  2. Drag to the diagram.
  3. Drop the OutputPort on the diagram.
  4. Select the OutputPort in the diagram.
  5. Open the Properties View.
  1. Select the Value cell for Name and type the OutputPort name (e.g. Calculated Velocity).
  2. Do not select the Value cell for Output Connection Policity. This value will be set once the port is connected to another port using a ConnectionPolicy.
  3. Selext the Value cell for Type and select from the drop down the C++ type of for the port. (e.g. float).
  4. Save your work.
