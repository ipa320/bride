Creating a model

  1. Select the model folder in your project.
  2. Click on File > New > Other...
  3. In the Wizards pane open the BRIDE folder and select RTT Model Diagram.
  4. Click Next.
  5. In the File Name box replace 'defualt' with the name of the diagram. Best Practice: Use the package name.
  6. Click Next (This step is mandatory if you want to have the diagram and the model to have the same name).
  7. Click Finish.
  8. Open the Propeties View and fill in the Name property for the diagram. Best Practice: Use the package name.
