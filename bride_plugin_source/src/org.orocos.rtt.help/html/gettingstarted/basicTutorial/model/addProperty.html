Adding a Property

This is how a Property looks like when first added to a TaskContext:

The easiest way to add a Property or any other model element is to select, drag and drop the element from the Palette to a TaskContext in the diagram.

  1. Select Property from the Palette.
  2. Drag the Property over the TaskContext and drop.
  3. Select the TaskContext.
  4. Select the Property.
  5. Open the Properties View.

  1. Select the Value cell for Description and type the optional description for the Property.
  2. Select the Value cell for Name and type the name for the Property.
  3. Select the Value cell for Type and select the type of the Property from the drop down list.
  4. Select the Value cell for Value and enter the value for the Property.
  5. Save your work.
