Adding an Operation

This is how a new Operation added to a TaskContext looks like:

The easiest way to add a Operation or any other model element is to select, drag and drop the element from the Palette to a TaskContext in the diagram.

  1. Select Operation from the Palette.
  2. Drag the Operation over the TaskContext and drop.
  3. Select the TaskContext.
  4. Select the Operation.
  5. Open the Properties View.

  1. Select the Value cell for Description and type the optional description for the Operation.
  2. Select the Value cell for Name and type the name for the Operation.
  3. Select the Value cell for Type and select the type of the Operation from the drop down list.
  4. Save your work.
