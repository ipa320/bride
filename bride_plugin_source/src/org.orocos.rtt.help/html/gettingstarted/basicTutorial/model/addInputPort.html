Adding an InputPort

This is how a TaskContext with an InputPort looks like when it is first created in the model diagram:

The easiest way to add a InputPort or any other model element is to select, drag and drop the element from the Palette to the diagram.

  1. Select a InputPort.
  2. Drag to the diagram.
  3. Drop the InputPort on the diagram.
  4. Select the InputPort in the diagram.
  5. Open the Properties View.

  1. Do not select the Value cell for Input Connection Policity. This value will be set once the port is connected to another port using a ConnectionPolicy.
  2. Select the Value cell for Is Event Port and choose between false and true from the drop down list.
  3. Select the Value cell for Name and type the InputPort name (e.g. Calculated Velocity).
  4. Selext the Value cell for Type and select from the drop down the C++ type of for the port. (e.g. float).
  5. Save your work.
