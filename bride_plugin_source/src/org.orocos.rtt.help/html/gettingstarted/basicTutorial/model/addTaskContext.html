Adding a TaskContext

This is how a TaskContext looks like when it is first created in the model diagram:

The easiest way to add a TaskContext or any other model element is to select, drag and drop the element form the Palette to the diagram.

  1. Select a TaskContext.
  2. Drag to the diagram.
  3. Drop the TaskContext on the diagram.
  4. Select the TaskContext in the diagram.
  5. Open the Properties View.

  1. Select the Value cell for Name and type the component name (e.g. Distance Sensor).
  2. Selcet the Value cell for Namespace and type the C++ namespace of for the TaskContext (e.g. same as package name).
  3. Selext the Value cell for Type and type the C++ type name. (e.g. DistanceSensorImpl).
  4. Save your work.
